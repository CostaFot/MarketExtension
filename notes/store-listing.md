# Microsoft Store listing copy

Compliance-reviewed copy for the Partner Center submission. Deliberately avoids:
no "real-time"/"live" guarantees (data can be delayed), no "free" anywhere, no implied
affiliation with Microsoft/providers/exchanges, no trading-advice language; required
disclaimers included.

## Description (main listing field)

**Markets Extension for Command Palette**

Track stocks, cryptocurrencies, and currencies right inside Microsoft PowerToys Command Palette — no separate app, no browser tab.

• Search symbols and view quotes and price charts
• Build a watchlist and mark favorites
• Track a portfolio you enter yourself, with multi-currency totals and daily change
• Pin favorites and your portfolio to the Command Palette dock
• Demo mode lets you explore with sample data, fully offline
• No accounts, no telemetry — it runs entirely on your machine

Connect the app using your own market-data API key from Twelve Data or Finnhub; currency reference rates need no key. Your keys and data stay on your device — the app has no servers and sends each key only to the provider it belongs to.

**Independent project.** Markets Extension is an open-source, independent extension. It is not affiliated with, endorsed by, or sponsored by Microsoft, Finnhub, Twelve Data, or any exchange or data provider. Company names, ticker symbols, and logos are the property of their respective owners.

**Disclaimer.** Market data is supplied by third parties and may be delayed or inaccurate. This app is provided for informational purposes only and is not financial, investment, tax, or trading advice, and is not a recommendation to buy or sell any asset. You are solely responsible for your own decisions; always do your own research.

## Compliance notes

- **No "real-time"/"live"** — sources can lag (ECB is daily; provider tiers delay), so a real-time claim would risk a misleading-functionality flag. Use "quotes and price charts" + the delay disclaimer.
- **Third-party account disclosure** — Store policy requires disclosing that a third-party service/key is needed; the "Connect the app using your own… key" line covers it.
- **Trademarks** — nominative use only ("for Microsoft PowerToys Command Palette"), plus an explicit non-affiliation + owners'-rights line.
- **Not financial advice** — explicit, since this is a markets app.
- **Never say the API keys are "free"** — provider tiers can change; keyless sources are described as "needs no key".
