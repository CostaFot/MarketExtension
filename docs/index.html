Privacy Policy

ADB Extension for Command Palette — Last updated April 2026

Overview

ADB Extension for Command Palette is a local developer tool that runs ADB (Android Debug Bridge) commands on your machine. It does not collect, transmit, or store any personal data.

Data Collection

This extension collects no data of any kind. It does not:

  • Connect to the internet
  • Send telemetry or analytics
  • Store any information outside of your local machine
  • Access files beyond what ADB requires to communicate with connected devices

ADB Communication

All communication happens locally between the extension and the ADB daemon running on your machine. No data leaves your device.
