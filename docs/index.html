Markets Extension for Command Palette

Live markets — right inside Windows PowerToys Command Palette.

Markets Extension is an open-source PowerToys Command Palette extension. You bring your own market-data API key; the extension runs entirely on your machine — no telemetry, no analytics, no accounts.
